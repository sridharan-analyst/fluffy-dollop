# HR Analytics Dashboard using Tableau & Excel

## Overview
This project analyzes employee attrition and workforce trends using the IBM HR Analytics dataset. The dashboard provides HR teams with actionable insights into employee retention, demographics, salary, and job satisfaction.

## Tools Used
- Tableau
- Microsoft Excel

## Dataset
IBM HR Analytics Employee Attrition Dataset

## Dashboard Features
- Employee Overview
- Attrition Analysis
- Department-wise Analysis
- Job Role Analysis
- Gender Analysis
- Age Group Analysis
- Salary Distribution
- Job Satisfaction Heat Map
- Interactive Filters

## Key KPIs
- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Monthly Income
- Average Years at Company

## Key Insights
- Sales department recorded the highest attrition.
- Employees aged 26–35 experienced the highest turnover.
- Employees with lower monthly income were more likely to leave.
- Shorter tenure was associated with higher attrition.

## Business Impact
The dashboard enables HR teams to identify employee turnover patterns and supports data-driven retention strategies.
